This example loads a csv file with randomly generated data. It contains following columns:

  • A, B, C, D which are only available in the 'DragDrop' area of the UI,
  • E which is entirely hidden,
  • x,y,e which are only available as arguments to aggregators,
  • z which is available as an argument for aggregator functions as well as within the 'DragDrop' area.
